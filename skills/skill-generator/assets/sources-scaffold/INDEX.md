# Реестр источников истины

Формат: `ID | тип | что отвечает | где лежит | обновлено`
Типы: DEC — решение; EXT — внешний указатель; CMD — команда получения выводимой правды.
Правило для агентов: нужен факт → найди строку здесь → прочитай источник → цитируй.
Строки нет → факта нет → спроси пользователя.
